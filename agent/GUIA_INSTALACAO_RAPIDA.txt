  🖨️  GUIA RÁPIDO - INSTALAR AGENTE EM OUTROS COMPUTADORES

📋 RESUMO: 3 PASSOS SIMPLES

1️⃣  COPIAR a pasta "agent" para o outro PC
2️⃣  INSTALAR as dependências (npm install)
3️⃣  CONFIGURAR e iniciar (npm run setup + npm start)

🎯 MÉTODO MAIS FÁCIL - USAR O INSTALADOR AUTOMÁTICO

NO SEU COMPUTADOR (onde está o projeto):
───────────────────────────────────────────────────────────────────

1. Copie a pasta inteira "agent" para um pendrive:
   
   📁 De: C:\Users\igor rodrigues\CascadeProjects\rotinas-ti-app\agent
   📁 Para: E:\agent (pendrive)


NO COMPUTADOR DE DESTINO (onde vai instalar):
───────────────────────────────────────────────────────────────────

1. Cole a pasta do pendrive no computador:
   
   📁 Para: C:\agent (ou qualquer lugar)

2. Entre na pasta e execute INSTALAR.bat:
   
   - Clique com botão direito em "INSTALAR.bat"
   - Escolha "Executar como administrador"
   - Siga as instruções na tela

3. Pronto! O instalador vai:
   ✅ Copiar arquivos para C:\printer-monitor-agent
   ✅ Instalar dependências automaticamente
   ✅ Perguntar se quer configurar agora

════════════════════════════════════════════════════════════════════

🎯 MÉTODO MANUAL (se preferir fazer passo a passo)

NO COMPUTADOR DE DESTINO:
───────────────────────────────────────────────────────────────────

1. COPIAR a pasta "agent" para:
   C:\printer-monitor-agent

2. ABRIR o CMD (Win + R → cmd → Enter)

3. NAVEGAR até a pasta:
   cd C:\printer-monitor-agent

4. INSTALAR dependências:
   npm install
   
   ⏳ Aguarde 2-3 minutos

5. CONFIGURAR o agente:
   npm run setup
   
   📝 Preencha:
   
   ┌─────────────────────────────────────────────────────────────┐
   │ Firebase Project ID:                                         │
   │ → rotinas-ti-hpaes                                          │
   │                                                              │
   │ Firebase Client Email:                                       │
   │ → firebase-adminsdk-xxxxx@rotinas-ti-hpaes.iam...          │
   │                                                              │
   │ Firebase Private Key:                                        │
   │ → Cole a chave completa (BEGIN...END)                       │
   │ → Pressione Enter DUAS VEZES                                │
   │                                                              │
   │ Nome deste computador:                                       │
   │ → PC da Recepção  ← DIFERENTE EM CADA PC!                  │
   │                                                              │
   │ Local/Setor:                                                │
   │ → Recepção  ← DIFERENTE EM CADA PC!                        │
   │                                                              │
   │ Intervalo (segundos):                                        │
   │ → 60 (ou Enter para padrão)                                 │
   │                                                              │
   │ Impressoras de rede? (s/N):                                 │
   │ → n (ou adicione se tiver)                                  │
   └─────────────────────────────────────────────────────────────┘

6. INICIAR o monitoramento:
   npm start
   
   ✅ Você verá:
   
   🚀 AGENTE DE MONITORAMENTO DE IMPRESSORAS INICIADO
   ✅ Configurações carregadas com sucesso
   📍 Computador: PC da Recepção
   🖨️  Detectadas X impressoras USB
   ✅ X/X impressoras sincronizadas com sucesso

🐛 PROBLEMAS COMUNS

"npm não é reconhecido"
───────────────────────────────────────────────────────────────────
→ Instale o Node.js: https://nodejs.org/


"Nenhuma impressora detectada"
───────────────────────────────────────────────────────────────────
→ Execute o CMD como Administrador
→ Verifique se há impressoras conectadas


"Erro ao inicializar Firebase"
───────────────────────────────────────────────────────────────────
→ Verifique as credenciais em config.json
→ Certifique-se de que copiou a chave completa


"Impressoras não aparecem no app web"
───────────────────────────────────────────────────────────────────
→ Aguarde 1 minuto para sincronização
→ Verifique se o agente está rodando (npm start)
→ Verifique conexão com internet
